《首页 推荐接口:

  • 保存推荐
  • 删除推荐
  • 获取推荐列表

保存推荐

method：POST
url: /api/recommend/save

参数列表

{
  "id": "int //推荐ID",
  "objectId": "long //对象ID",
  "position": "int //排序",
  "objectType": "byte //对象类型"
}

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": {}
}

删除推荐

method：POST
url: /api/recommend/delete

参数列表

参数名类型必需描述
idinttrue推荐ID

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",

获取推荐列表

method：GET
url: /api/recommend/list

参数列表

参数名类型必需描述
pageintfalse页数
limitintfalse每页条数
sortstringfalse排序
objectTypebytefalsenull

{

    "pageCount": "int //页数",
    "currentPage": "int //当前页",
    "pageSize": "int //每页记录数",

      {
        "id": "int",
        "position": "int",

      }
    ],
    "hasMore": "boolean //是否还有更多"
  }
