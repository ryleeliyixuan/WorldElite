《首页 配置接口:

  • 保存首页文案配置
  • 获取首页文案配置
  • 用户协议

保存首页文案配置

method：POST
url: /api/config/save-home-config

参数列表

{
  "configType": "int //配置类型",
  "mainTitle": "string //主标题",
  "subTitle": "string //副标题",
  "picUrl": "string //图片"
}

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": {}
}

获取首页文案配置

method：GET
url: /api/config/get-home-config

参数列表

参数名类型必需描述
configTypeinttrue配置类型

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": {
    "configType": "int //配置类型",
    "mainTitle": "string //主标题",
    "subTitle": "string //副标题",
    "picUrl": "string //图片"
  }
}
