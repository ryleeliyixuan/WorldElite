《首页 国家数据接口:

  • 获取所有国家代码
  • 获取当前IP所在的国家代码

获取所有国家代码

method：GET
url: /api/country/get-all

参数列表

参数名类型必需描述

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": [
    {
      "id": "int //ID",
      "phoneCode": "string //电话区号",
      "countryCode": "string //国家代号",
      "chineseName": "string //中文名",
      "englishName": "string //英文名"
    }
  ]
}
