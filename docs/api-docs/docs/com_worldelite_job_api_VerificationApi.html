《首页 企业验证接口:

  • 提交验证信息
  • 获取我的验证信息
  • 获取审核资料
  • 企业用户审核通过或者拒绝

提交验证信息

method：POST
url: /api/verify/save-verify-info

参数列表

{
  "name": "string //名字",
  "idCard": "string //身份证号",
  "idCardPic": "string //身份证图片",
  "company": "string //公司",
  "post": "string //职位",
  "companyLicensePic": "string //营业执照"
}

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": {}
}

获取我的验证信息

method：GET
url: /api/verify/my-verify-info

参数列表

参数名类型必需描述

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": {
    "name": "string //名字",
    "idCard": "string //身份证号",
    "idCardPic": "string //身份证图片",
    "company": "string //公司",
    "post": "string //职位",
    "companyLicensePic": "string //营业执照",
    "remark": "string //备注",
    "status": "byte //状态"
  }
}

企业用户审核通过或者拒绝

method：POST
url: /api/verify/verify-company-user

参数列表

{
  "userId": "long //用户ID",
  "status": "byte //状态",
  "reason": "string //原因"
}

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": {}
}
