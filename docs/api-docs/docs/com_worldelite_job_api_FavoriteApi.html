《首页 收藏接口:

  • 收藏或者取消收藏
  • 获取当前用户收藏列表
  • 当前用户参与活动列表

收藏或者取消收藏

method：POST
url: /api/favorite/favorite

参数列表

{
  "objectId": "long //对象ID",
  "type": "byte //对象类型",
  "favorite": "boolean //是否已收藏"
}

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": {}
}

获取当前用户收藏列表

method：GET
url: /api/favorite/my-favorite-list

参数列表

参数名类型必需描述
pageintfalse页数
limitintfalse每页条数
sortstringfalse排序
typebytefalse对象类型

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": {}
}

当前用户参与活动列表

method：GET
url: /api/favorite/my-favorite-activities

参数列表

参数名类型必需描述
pageintfalse页数
limitintfalse每页条数
sortstringfalse排序

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": {
    "total": "int //总记录数",
    "pageCount": "int //页数",
    "currentPage": "int //当前页",
    "pageSize": "int //每页记录数",
    "list": [
      {
        "id": "int //活动ID",
        "title": "string //活动标题",
        "thumbnail": "string //缩略图",
        "summary": "string //摘要",
        "url": "string //链接",
        "description": "string //活动详情",
        "city": {
          "id": "int //字典ID",
          "name": "string //字典名",
          "type": "int //字典类型",
          "value": "string //字典值"
        },
        "startTime": "date //开始时间",
        "finishTime": "date //结束时间",
        "address": "string //活动地址",
        "status": "byte //状态",
        "joinFlag": "boolean //是否参加",
        "joinTime": "date //参加时间"
      }
    ],
    "hasMore": "boolean //是否还有更多"
  }
}
