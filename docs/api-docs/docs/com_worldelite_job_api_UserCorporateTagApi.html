《首页 企业标签操作接口:

  • 保存用户标签
  • 获取标签列表
  • 删除标签

保存用户标签

method：POST
url: /api/usercorporate/tag/save

参数列表

{
  "id": "long",
  "jobApplyId": "long",
  "tagName": "string"
}

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": {}
}

获取标签列表

method：GET
url: /api/usercorporate/tag/list

参数列表

参数名类型必需描述
jobApplyIdlongtrue

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": [
    {
      "id": "long //标签ID",
      "jobApplyId": "long //职位申请ID",
      "tagName": "string //企业给个人用户设置的标签",
      "createTime": "date //标签创建时间"
    }
  ]
}
