《首页 用户笔记操作接口:

  • 保存用户笔记
  • 获取某用户的所有笔记
  • 删除用户笔记

保存用户笔记

method：POST
url: /api/usercorporate/comment/save

参数列表

{
  "id": "long",
  "jobApplyId": "long",
  "comment": "string"
}

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": {}
}

获取某用户的所有笔记

method：GET
url: /api/usercorporate/comment/list

参数列表

参数名类型必需描述

返回结果: Android Code | IOS Code

{
  "code": "int //0为成功，其他错误码参考 ApiCode",
  "msg": "string //错误消息",
  "data": [
    {
      "id": "long //笔记Id， Primary Key",
      "jobApplyId": "long //职位Id",
      "comment": "string //用户笔记",
      "createTime": "date //创建时间"
    }
  ]
}
